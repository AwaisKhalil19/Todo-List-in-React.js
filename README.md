# 📝 Todo List App

A simple and responsive Todo List application built with React.js, JavaScript, and Tailwind CSS. This app helps users manage daily tasks efficiently by adding, completing, and deleting todos.

## 🚀 Features

- ✅ Add new tasks
- ✏️ Mark tasks as completed
- 🗑️ Delete tasks
- 📱 Fully responsive design
- ⚡ Fast and user-friendly interface
- 🎨 Modern UI with Tailwind CSS
- 🔥 React Icons integration

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- React Icons
